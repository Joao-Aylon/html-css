Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag b (Não semântica).

Nesta frase, tenho um termo em destaque usando a tag STRONG (Semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (Não semântica).

Outra maneira de usar o itálico é: aperte 'ctrl' + 'shift' + 'p'. Dígite 'abb' e selecione a tecla 'i'.

Nesta frase, temos um termo em ênfase usando a tag Em (Semântica).

Texto Marcado

Podemos criar também um texto marcado usando a Tag MARK.
Podemos trocar a cor da marcação do texto usando o recurso Style do CSS.

Texto grande e Pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.Para usar texto pequeno, usa-se a tag SMALL(Semântica).

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.Usa-se a tag DEL(Semântica).

Texto excluído

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.Usa-se a tag INS(Semântica).

Existe também o sublinhado com a tag U (Nã semântica).

Texto Sobrescrito

Para inserir coisas do tipo x25+5. Usando a tag SUP(Semântica).

Texto Subscrito

Para inserir coisas do tipo H2O. Usando a tag SUB (Semântica).
